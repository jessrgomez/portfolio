Web Developement Aerovault Tracking System (ATS)

ATS is a specialized aviation system designed to streamline and simplify the management of leased asset documentation between airlines and lessors. With two core modules Records Management and Project Management the system enhances operational efficiency while ensuring regulatory compliance throughout the leased asset lifecycle. Records Management Module: Allows users to organize, view, search, retrieve, download, and transfer documents digitally, all while supporting records compliance requirements. Project Management Module: Enables users to create and manage projects, view interactive dashboards for project status, track multiple projects in parallel, and manage documentation checklists in real time. Built with React.js, Redux, Axios, and Node.js, and styled using Material UI. Storybook is used for UI component development and documentation.
